## Overview
This project implements a medical question-answering chatbot using a
sample of the MedQuAD dataset. The chatbot retrieves relevant answers
based on user queries and performs basic medical entity recognition.

## Features
- Medical question answering
- TF-IDF based retrieval mechanism
- Basic medical entity recognition
- Streamlit web interface

## Dataset
A small sample of the MedQuAD dataset is used to demonstrate the system.

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the application:
   streamlit run app.py
